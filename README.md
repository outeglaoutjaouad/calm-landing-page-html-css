# Mini Projet HTML & CSS – Landing Page Calm

Ce mini projet consiste à recréer une landing page inspirée du site Calm en utilisant uniquement **HTML** et **CSS**.  
L’objectif est de pratiquer la structuration d’une page web, le responsive design et la mise en forme moderne (navbar, hero, sections, cartes, footer, etc.).

La page contient plusieurs sections :  
- Une barre de navigation  
- Une section Hero  
- Des cartes de fonctionnalités  
- Une galerie horizontale  
- Des témoignages utilisateurs  
- Un bloc de tarification (pricing)  
- Un footer complet avec liens et icônes sociales  

Ce projet permet de renforcer les bases du front-end tout en reproduisant un design professionnel.

## Technologies utilisées
- **HTML5** – structure de la page  
- **CSS3** – mise en forme, flexbox, grid, responsive design  

Aucune bibliothèque ou framework n’a été utilisé.  
Le projet est 100 % **HTML + CSS**.
